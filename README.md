## Overview
This repository is a collection of artificial intelligence projects, including puzzle solvers, game implementations, and algorithm demonstrations. It is designed for developers and AI enthusiasts who want to explore and learn from various AI concepts and techniques. The projects in this repository are implemented in Java and cover a range of topics, from classic puzzles like Sudoku and Magic Square to more complex problems like the N-Queens problem and the Missionaries and Cannibals problem.

## ‚öôÔ∏è Installation
To install and run the projects, follow these steps:
1. Install the Java Development Kit (JDK) on your system
2. Clone the repository using the command: git clone https://github.com/jungharepranay/AI.git
3. Navigate to the project folder and compile the Java files using the command: javac *.java
4. Run the projects using the command: java <ProjectName>

## üß™ Usage
To use the projects, simply run the corresponding Java file. For example, to run the Sudoku puzzle solver, use the command: java Sudoku. The projects will generate solutions or play games according to their specific implementation.
